## 📱 Android App

MovieRater is also available as a native Android app!

**[⬇️ Download the latest APK](https://github.com/Michelle-Mungai/movierater/releases/latest)**

### Installation
1. Download the APK from the link above
2. On your Android device, open the downloaded file
3. If prompted, allow **"Install from unknown sources"** for your browser/file manager
4. Tap **Install**

> Note: Since this app isn't distributed via Google Play, Android will show a warning before installing. This is expected — the app is safe, it's just not going through Play Store review.

### Features
- Browse and search movies
- Rate and review titles
- Get personalized recommendations based on your ratings
- Track your review history
